# CodeAlpha Simple E-Commerce Store

Full Stack Development Internship Task 1.

## Features
- User registration and login
- Product listings
- Product details
- Shopping cart
- Order processing
- Order history
- MongoDB database for users, products and orders

## Tech Stack
HTML, CSS, JavaScript, Node.js, Express.js, MongoDB

## Run
1. Install Node.js and MongoDB.
2. Start MongoDB.
3. Run `npm install`
4. Run `npm start`
5. Open `http://localhost:3000`
